### 문제 설명

<p>10진법 수 N이 주어진다. 이 수를 B진법으로 바꿔 출력하는 프로그램을 작성하시오.</p>

<p>10진법을 넘어가는 진법은 숫자로 표시할 수 없는 자리가 있다. 이런 경우에는 다음과 같이 알파벳 대문자를 사용한다.</p>

<p>A: 10, B: 11, ..., F: 15, ..., Y: 34, Z: 35</p>

### 입력 

 <p>첫째 줄에 N과 B가 주어진다. (2 ≤ B ≤ 36) N은 10억보다 작거나 같은 자연수이다.</p>

### 출력 

 <p>첫째 줄에 10진법 수 N을 B진법으로 출력한다.</p>

### 풀이 과정
1. 자바는 배열을 한 번 선언하면 크기를 바꿀 수 없으므로 N을 B로 나누며 B진법으로 변환했을때의 자릿수를 확인한다.
2. 배열을 선언하고 N을 B로 나눈 나머지를 넣는다.
3. 각 나머지를 10진법으로 변환해서 출력한다.
3-1. 10보다 크면 아스키코드를 이용하여 문자로 출력한다.

### 배운 것
- 배열의 크기를 바꿀 수 없으므로 배열의 크기가 경우마다 다를땐 미리 배열의 크기를 측정해서 선언하는 과정이 필요하다는 것을 알게 되었다.
